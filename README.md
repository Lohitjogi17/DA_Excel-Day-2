## 1. Font & Styling

## 2. Alignment & Cell Management
Ensuring data is positioned correctly for better scanning:

Text Positioning: Using Top, Middle, Bottom, and Left/Right alignments.

Wrap Text: Managing long strings of text within a single cell to maintain column width.

Merge & Center: Creating clean headers across multiple columns for report titles.

## 3. Number Formatting & Data Types
This is the most critical part of data integrity. I explored how Excel treats different data types:

General & Number: Standard numerical data.

Currency & Accounting: Proper formatting for financial values (using $ or other symbols).

Dates: Switching between Short and Long date formats to ensure chronological clarity.

Percentages & Fractions: Representing ratios accurately.
